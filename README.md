Ce projet est découpé en 3 activités : 

Cours : vous suivrez le cours Construisez un site web à l’aide du framework Symfony 5 pour découvrir ce framework.

Exercice : vous réaliserez l’exercice Faites vos premiers pas avec Symfony pour réaliser un site web en utilisant ce framework.

À l’issue de cette activité, vous ferez une auto-évaluation de votre travail ainsi qu’une session de bilan avec votre mentor.

Mission : vous réaliserez la mission Créez une plateforme de gestion de projet avec Symfony pour réaliser un site web plus complet, et en autonomie, en utilisant ce framework.

À l’issue de cette activité, vous ferez une auto-évaluation de votre travail ainsi qu’une session de bilan avec votre mentor.
